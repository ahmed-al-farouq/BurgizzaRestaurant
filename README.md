# Burgizza Restaurant

> Burgizza Restaurant is a site that provides all kinds of burgers and pizzas

## Built With
  - NextJS
  - Redux
  - Redux-Persist
  - React-Icons
  - Formik & Yup
  - Axios
  - Custom API

## Live Demo
[Live-demo](https://burgizza.pages.dev/)
### Prerequisites
  - #### Knowledge of these technologies
    - NextJS
    - Redux
### Setup
  * Open your command
  * Type "git clone https://github.com/ahmed-al-farouq/BurgizzaRestaurant.git"
  * npm i
  * npm run build
  * npm run start
### Run the app
  * npm run start

### Deployment
  * npm run build
